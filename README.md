# Anime-List-Site
Anime Site For Discovering New Anime and Keeping Track of Watched Anime

### Technologies Used
- NodeJS
- Express
- MongoDB
- Bootstrap
- EJS

### Current Features
- Add Animes
- Delete Animes
- Edit Animes
- Leave Reviews on an Anime

### Features Being Worked On
- User Authentication
- Authorization
- Cookies/Sessions
- Website design
